As with many projects in Python, I start with the Hitchhiker's Guide to Python. Here are two quotes from the site on picking a web framework:

So from a quick start and learning perspective, I'm using Flask - purely a learning exercise. I'm also going to use VS Code for the project so screenshots and info will be from there. You can download VS Code for free here.

Getting up and running

You'll want a python virtualenv before you start. Since I'm running on an Anaconda installation on windows, I'm going to install flask with a conda command, rather than pip. Both are below:

Conda:

conda install flask

Pip:

pip install flask

Now that flask is installed, I loaded up a new folder for scripts in VS Code, and created a main.py with this block (straight from the Flask docs):

from flask import Flask
app = Flask(__name__)

@app.route("/")
def hello():
    return "Hello World!"

if __name__ == "__main__":
    app.run()
